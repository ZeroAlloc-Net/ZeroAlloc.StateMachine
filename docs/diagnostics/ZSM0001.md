# ZSM0001 — Unreachable State

**Severity:** Warning

A state appears in a `From` position in at least one transition but no transition leads to it (it is not in any `To` position) and it is not the `InitialState`. The state is unreachable at runtime.

## Example

```csharp
// ZSM0001: Orphan is unreachable — nothing transitions to it
[StateMachine(InitialState = nameof(S.Start))]
[Transition<S, R>(From = S.Start,  On = R.Go,     To = S.End)]
[Transition<S, R>(From = S.Orphan, On = R.Detour, To = S.End)]
public partial class MyMachine { }
```

## How to fix

Add a transition that leads to the unreachable state, or remove its outgoing transitions if it was declared by mistake.
