# ZSM0002 — Unintentional Sink State

**Severity:** Warning

A state appears in a `To` position (something transitions into it) but has no outgoing transitions, and it is not annotated `[Terminal<TState>]`. The machine will be stuck in this state with no way to leave.

## Example

```csharp
// ZSM0002: End has no outgoing transitions and is not [Terminal]
[StateMachine(InitialState = nameof(S.Start))]
[Transition<S, R>(From = S.Start, On = R.Go, To = S.End)]
public partial class MyMachine { }
```

## How to fix

If `End` is an intentional terminal state, annotate it:

```csharp
[Terminal<S>(State = S.End)]
```

Otherwise, add an outgoing transition from `End`.
